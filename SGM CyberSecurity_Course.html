Course Snapshot

Duration: 40–60 hours

Level: Beginner → Advanced

Mode: Classroom / Online

Course Modules

  • Understanding Cybersecurity and its importance
  • Types of cyber threats and attacks
  • Overview of cybersecurity frameworks and standards

  • Basics of networking: LAN, WAN, IP addressing
  • Understanding protocols: TCP/IP, HTTP, HTTPS, FTP, DNS
  • Firewalls, routers, and network devices overview

  • Windows and Linux security basics
  • User and permission management
  • System hardening and patch management

  • Identifying vulnerabilities in systems and networks
  • Risk assessment and mitigation strategies
  • Security policies and compliance

  • Types of malware: viruses, worms, ransomware, spyware
  • Social engineering attacks: phishing, baiting, impersonation
  • Exploits, trojans, and zero-day attacks

  • Introduction to ethical hacking principles
  • Footprinting, scanning, and enumeration techniques
  • Vulnerability assessment and penetration testing

  • Configuring firewalls and intrusion detection/prevention systems (IDS/IPS)
  • VPNs and secure remote access
  • Network monitoring tools and threat detection

  • Web application vulnerabilities (SQL Injection, XSS, CSRF)
  • Securing web applications and databases
  • Secure coding practices

  • Cybersecurity in Cloud and Virtual Environments
  • Encryption, cryptography, and secure communication
  • Incident response and digital forensics

  • Hands-on exercises for real-world cyber threats
  • Simulated cyber attack and defense scenarios
  • Preparation for cybersecurity certifications like CompTIA Security+, CEH
